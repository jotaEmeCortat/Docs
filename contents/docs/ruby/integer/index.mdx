## Methods for Creating

```ruby
Integer.new(1)                     # => 1
int = 2                            # => 2
```

## Methods for Querying

### allbits?

Returns whether **all** bits in `self` are set.

```ruby
0b1010101.allbits?(0b1010100)      # => true
0b1010100.allbits?(0b1010101)      # => false
```

### anybits?

Returns whether **any** bits in `self` are set

```ruby
0b10000010.anybits?(0b11111111)    # => true
0b00000000.anybits?(0b11111111)    # => false
```

### between?

Returns whether `self` is between the given values

```ruby
2.between?(1, 3)                   # => true
2.between?(3, 5)                   # => false
```

### eql?

Returns whether `self` is equal to the given value

```ruby
1.eql?(1)                          # => true
1.eql?(1.0)                        # => false
```

### equal

Returns whether `self` is equal to the given value

```ruby
1 == 1                             # => true
1 == 1.0                           # => true
```

### even?

Returns whether `self` is an even number

```ruby
2.even?                            # => true
3.even?                            # => false
```

### greater

Returns whether `self` is greater than the given value

```ruby
2 > 1                              # => true
1 > 2                              # => false
```

### greater or equal

Returns whether `self` is greater than or equal to the given value

```ruby
2 >= 1                             # => true
1 >= 2                             # => false
```

### integer?

Returns whether `self` is an Integer

```ruby
1.integer?                         # => true
1.0.integer?                       # => false
```

### less

Returns whether `self` is less than the given value

```ruby
1 < 2                              # => true
2 < 1                              # => false
```

### less or equal

Returns whether `self` is less than or equal to the given value

```ruby
1 <= 2                             # => true
2 <= 1                             # => false
```

### negative?

Returns whether `self` is a **negative** number

```ruby
-1.negative?                       # => true
1.negative?                        # => false
```

### nobits?

Returns whether **no** bits in `self` are set

```ruby
0b11110000.nobits?(0b00001111)     # => true
0b00000001.nobits?(0b11111111)     # => false
```

### odd?

Returns whether `self` is an odd number

```ruby
2.odd?                             # => false
3.odd?                             # => true
```

### positive?

Returns whether `self` is a **positive** number

```ruby
-1.positive?                       # => false
1.positive?                        # => true
```

### zero?

Returns whether `self` is zero

```ruby
0.zero?                            # => true
1.zero?                            # => false
```

## Methods for Fetching

### [ ]

Returns a slice of bits from `self`

```ruby
0b1010101[0]                       # => 1
0b1010101[1]                       # => 0
```

## Methods for Iterating

### downto

Iterates over the given block, passing decreasing values from `self` down to the given value

```ruby
a = []
5.downto(1) { |i| print i, " " }   # => 5 4 3 2 1
4.downto(2) { |i| i << a }         # => [4, 3, 2]
```

### times

Calls the given block `self`times with each integer in `(0..self-1)`

```ruby
5.times { |i| print i, " " }       # => 0 1 2 3 4
```

### upto

Calls the given block with each integer value from `self` up to the given value

```ruby
a = []
1.upto(5) { |i| print i, " " }     # => 1 2 3 4 5
2.upto(4) { |i| i << a }           # => [2, 3, 4]
```

## Methods for Converting

### *

Returns the product of `self` and the given value

```ruby
4 * 2                              # => 8
4 * -2                             # => -8
4 * 2.0                            # => 8.0
```

### **

Returns the value of `self` raised to the power of the given value

```ruby
2 ** 3                             # => 8
2 ** 3.3                           # => 9.849155306759329
```

### +

Returns the sum of `self` and the given value

```ruby
2 + 3                              # => 5
2 + 2.0                            # => 4.0
```

### -

Returns the difference of `self` and the given value

```ruby
2 - 3                              # => -1
4 - 2.0                            # => 2.0
```

### /

Returns the quotient of `self` and the given value

```ruby
4 / 3                              # => 1
4 / 2                              # => 2
```

### %

Returns `self` modulo the given value

```ruby
10 % 2                             # => 0
10 % -2                            # => 0
10 % 3.0                           # => 1.0
```

### abs

Returns the absolute value of `self`

```ruby
-12345.abs                         # => 12345
12345.abs                          # => 12345
```

### ceil

Returns the smallest number **greater** than or equal to `self`

```ruby
1.ceil                             # => 1
4.2.ceil                           # => 5
```

### ceildiv

Returns the integer result of dividing `self` by the given value

```ruby
5.ceildiv(3)                       # => 2
4.ceildiv(2)                       # => 2
```

### chr

Returns a 1-character string containing the character represented by the value of `self`

```ruby
65.chr                             # => "A"
0.chr                              # => "\x00"
```

### denominator

Returns the denominator of `self`

```ruby
1.denominator                      # => 1
1.5.denominator                    # => 2
```

### digits

Returns an array of integers representing the base-radix digits of `self`

```ruby
12345.digits                       # => [5, 4, 3, 2, 1]
12345.digits(7)                    # => [3, 3, 3, 3, 1, 0]
```

### div

Returns the integer result of dividing `self` by the given value

```ruby
5.div(3)                           # => 1
-4.div(3)                          # => -2
```

### divmod

Returns a 2-element array containing the quotient and remainder results of dividing `self` by the given value

```ruby
5.divmod(3)                        # => [1, 2]
4.divmod(2)                        # => [2, 0]
```

### fdiv

Returns the floating-point result of dividing `self` by the given value

```ruby
5.fdiv(3)                          # => 1.6666666666666667
4.fdiv(2)                          # => 2.0
```

### floor

Returns the greatest number **smaller** than or equal to `self`

```ruby
1.floor                            # => 1
4.2.floor                          # => 4
-2.7.floor                         # => -3
```

### gcd

Returns the greatest **common** divisor of `self` and the given value

```ruby
4.gcd(2)                           # => 2
3.gcd(-7)                          # => 1
```

### gcdlcm

Returns a 2-element array containing the **greatest** common divisor and **least** common multiple of `self` and the given value

```ruby
4.gcdlcm(2)                        # => [2, 4]
3.gcdlcm(-7)                       # => [1, 21]
```

### lcm

Returns the **least** common multiple of `self` and the given value

```ruby
4.lcm(2)                           # => 4
3.lcm(-7)                          # => 21
```

### pow

Returns the **modular** **exponentiation** of `self`

```ruby
2.pow(3, 5)                        # => 3
```

### pred

Returns the integer **predecessor** of `self`

```ruby
1.pred                             # => 0
2.pred                             # => 1
```

### rationalize

Returns the `Rational` representation of `self`

```ruby
1.rationalize                      # => (1/1)
1.5.rationalize                    # => (3/2)
```

### remainder

Returns the remainder **after** **dividing** `self` by the given value

```ruby
5.remainder(3)                     # => 2
4.remainder(2)                     # => 0
```

### round

Returns `self` rounded to the nearest value with the given precision

```ruby
4.2.round                          # => 4
-4.2.round                         # => -4
```

### sqrt

Returns the integer square root of the given value.

```ruby
Integer.sqrt(0)                    # => 0
Integer.sqrt(4.0)                  # => 2
Integer.sqrt(24)                   # => 4

Integer.sqrt(10**400)              # => 10**200
```

### succ

Returns the integer successor of `self`

```ruby
1.succ                             #=> 2
-1.succ                            #=> 0
```

### to_f

Returns `self` converted to a `Float`

```ruby
1.to_f                             # => 1.0
```

### to_s

Returns a `String` containing the place-value representation of `self` in the given radix

```ruby
12345.to_s                         # => "12345"
12345.to_s(2)                      # => "11000000111001"
```

### truncate

Returns `self` truncated to the given precision

```ruby
4.2.truncate                       # => 4
-4.2.truncate                      # => -4
```

### try_convert

Returns the given value converted to an Integer

```ruby
Integer.try_convert(1.25)          # => 1
Integer.try_convert("1")           # => 1
Integer.try_convert("a")           # => nil
```
